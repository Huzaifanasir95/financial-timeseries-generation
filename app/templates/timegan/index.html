TimeGAN Loss Functions

1. Reconstruction Loss (L_R)

Ensures meaningful embeddings by minimizing reconstruction error:

L_R = E[||x - r(e(x))||²]

Where e() is embedder, r() is recovery network

2. Supervised Loss (L_S)

Preserves temporal conditional distributions:

L_S = Σ ||h_t - g(e(x_{<t}), z_t)||²

Stepwise prediction ensures temporal coherence

3. Adversarial Loss (L_A)

Matches joint distributions via minimax game:

L_A = E[log d(e(x))] + E[log(1 - d(g(z)))]

Adversarial training in latent space

Total Loss
L_total = 10·L_R + 0.1·L_S + 1·L_A

Weighted combination of all three objectives
